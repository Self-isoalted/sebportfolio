Week 1

Some people may learn a little faster at the beginning, but in the long run the most important factor is practice and persistence. The more problems you solve, the more natural mathematics becomes.

The exercises below start with simple questions and gradually become more challenging. Try to solve them on your own first. If you get stuck, think about the ideas we discussed during the lesson and review the examples above.

Remember: making mistakes is part of learning. What matters is understanding why the correct solution works.

Exercise 1

Which of these is an improper fraction?

  • \(\frac{2}{7}\)
  • \(\frac{5}{5}\)
  • \(\frac{7}{4}\)
  • \(\frac{3}{8}\)
Show solution

An improper fraction has a numerator greater than the denominator.

\[ \frac{7}{4} \]

is the improper fraction.

Exercise 2

Work out:

\[ \frac{1}{4} + \frac{2}{4} \]
Show solution

The denominators are already the same, so add the numerators.

\[ \frac{1}{4} + \frac{2}{4} = \frac{3}{4} \]

Exercise 3

Work out:

\[ \frac{5}{6} - \frac{2}{6} \]
Show solution

The denominators are the same, so subtract the numerators.

\[ \frac{5}{6} - \frac{2}{6} = \frac{3}{6} \]

Simplify the result:

\[ \frac{3}{6} = \frac{1}{2} \]

Exercise 4

Find the least common denominator and work out:

\[ \frac{1}{2} + \frac{1}{4} \]
Show solution

The least common denominator is 4.

\[ \frac{1}{2} = \frac{2}{4} \]

Now add:

\[ \frac{2}{4} + \frac{1}{4} = \frac{3}{4} \]

Exercise 5

Find the least common denominator and work out:

\[ \frac{3}{4} - \frac{1}{8} \]
Show solution

The least common denominator is 8.

\[ \frac{3}{4} = \frac{6}{8} \]

Now subtract:

\[ \frac{6}{8} - \frac{1}{8} = \frac{5}{8} \]

Exercise 6

Multiply:

\[ \frac{2}{3} \times \frac{4}{5} \]
Show solution

Multiply the numerators and multiply the denominators.

\[ \frac{2}{3} \times \frac{4}{5} = \frac{2 \times 4}{3 \times 5} = \frac{8}{15} \]

Exercise 7

Multiply:

\[ \frac{3}{7} \times 4 \]
Show solution

Write the whole number as a fraction:

\[ 4 = \frac{4}{1} \]

Now multiply:

\[ \frac{3}{7} \times \frac{4}{1} = \frac{12}{7} \]

Exercise 8

Divide:

\[ \frac{2}{3} \div \frac{4}{5} \]
Show solution

Keep the first fraction, flip the second fraction, and change to multiplication.

\[ \frac{2}{3} \div \frac{4}{5} = \frac{2}{3} \times \frac{5}{4} \]

Now multiply:

\[ \frac{2 \times 5}{3 \times 4} = \frac{10}{12} = \frac{5}{6} \]

Exercise 9

Divide:

\[ \frac{3}{4} \div \frac{2}{5} \]
Show solution \[ \frac{3}{4} \div \frac{2}{5} = \frac{3}{4} \times \frac{5}{2} = \frac{15}{8} \]

Exercise 10

Work out:

\[ \frac{1}{2} + \frac{1}{3} + \frac{1}{6} \]
Show solution

The least common denominator is 6.

\[ \frac{1}{2} = \frac{3}{6} \] \[ \frac{1}{3} = \frac{2}{6} \] \[ \frac{1}{6} = \frac{1}{6} \]

Now add:

\[ \frac{3}{6} + \frac{2}{6} + \frac{1}{6} = \frac{6}{6} = 1 \]

Exercise 11

Work out:

\[ \frac{3}{5} - \frac{1}{10} + \frac{2}{5} \]
Show solution

The least common denominator is 10.

\[ \frac{3}{5} = \frac{6}{10} \] \[ \frac{2}{5} = \frac{4}{10} \]

Now calculate:

\[ \frac{6}{10} - \frac{1}{10} + \frac{4}{10} = \frac{9}{10} \]

Exercise 12

Work out:

\[ \frac{2}{3} \times \frac{9}{4} \]
Show solution

You can multiply directly:

\[ \frac{2}{3} \times \frac{9}{4} = \frac{18}{12} \]

Simplify:

\[ \frac{18}{12} = \frac{3}{2} \]

Or simplify first if you prefer.

Exercise 13

Work out:

\[ \frac{4}{7} \div 2 \]
Show solution

Write 2 as a fraction:

\[ 2 = \frac{2}{1} \]

Now divide by multiplying by the reciprocal:

\[ \frac{4}{7} \div \frac{2}{1} = \frac{4}{7} \times \frac{1}{2} = \frac{4}{14} = \frac{2}{7} \]

Exercise 14

Challenge question:

\[ \frac{1}{2} \div \frac{1}{4} \]
Show solution

Flip the second fraction and multiply:

\[ \frac{1}{2} \div \frac{1}{4} = \frac{1}{2} \times \frac{4}{1} = \frac{4}{2} = 2 \]

This makes sense: there are two quarters in one half.

Exercise 15

Challenge question:

\[ \left(\frac{1}{2} + \frac{1}{4}\right) \times \frac{2}{3} \]
Show solution

First work out the brackets:

\[ \frac{1}{2} + \frac{1}{4} = \frac{2}{4} + \frac{1}{4} = \frac{3}{4} \]

Now multiply:

\[ \frac{3}{4} \times \frac{2}{3} = \frac{6}{12} = \frac{1}{2} \]

Exercise 16

Final challenge:

\[ \frac{3}{4} \div \frac{1}{2} + \frac{1}{8} \]
Show solution

First do the division:

\[ \frac{3}{4} \div \frac{1}{2} = \frac{3}{4} \times \frac{2}{1} = \frac{6}{4} = \frac{3}{2} \]

Now add \(\frac{1}{8}\):

\[ \frac{3}{2} = \frac{12}{8} \] \[ \frac{12}{8} + \frac{1}{8} = \frac{13}{8} \]
